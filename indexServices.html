Mobile Dev

A mobile developer is a professional who is focused on developing software or systems for mobile devices

Desktop Dev

A desktop developer is a programmer who writes code for software applications that (1) run natively on operating systems like macOS, Windows, and Linux, and (2) don't need to be connected to the internet.

Web Dev

A Web developer is usually concerned with the back end or the programming aspect of creating a website or Web application and is not to be confused with a Web designer.

Architect

a person who designs hardware, software, or networking applications and services of a specified type for a business or other organization.
